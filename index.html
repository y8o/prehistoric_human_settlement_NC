Prehistoric Settlement Analysis

This spatial analysis investigates whether proximity to streams influenced where prehistoric humans settled, using archaeological site data near Fayetteville, NC.

Tools Used

ArcGIS Buffer Analysis Minimum Bounding Geometry Spatial Join Clip and Intersect Chi-Square Test

Key Findings

  • Most sites were within 100 meters of streams.
  • Statistically significant results (p < 0.05) show non-random site distribution.
  • Both projectile points and end scrapers show spatial bias near water.
  • Human activity was likely influenced by access to water and resources.

Map Layers

  • Elevation Raster (DEM)
  • Projectile Point Sites
  • End Scraper Sites
  • Clipped Stream Buffers (0–500+ m)
